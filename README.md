# adv-comp-arch-hw2
Advanced Computer Architecture Homework 2 project for Syracuse University

#### OPCODE LIST:
- **\+** : add
- **+n** : add number
- **\-** : subtract
- **==** : branch if equal
- **!=** : branch if not equal
- **go** : go to address
- **=n** : set to number
- **<** : set to if less than
- **<n** : set to if less than number
- **end** : end instructions

#### REGISTER LIST:
- $0
- $1
- $2
- $3
- $4
